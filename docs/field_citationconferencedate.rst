30. Citation Conference Date (R)
================================

OAIRE Field
~~~~~~~~~~~
``oaire:citationConferenceDate``

Usage
~~~~~

The date when the conference took place

Example
~~~~~~~

.. code-block:: xml
   :linenos:

   <oaire:citationConferenceDate>September 22-26, 2013</oaire:citationConferenceDate>
