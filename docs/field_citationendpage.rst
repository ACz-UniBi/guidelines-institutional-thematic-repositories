27. Citation End Page (O)
=========================

OAIRE Field
~~~~~~~~~~~
``oaire:citationEndPage``

Usage
~~~~~

The end page of the work published in a container (e.g. journal issue)

Example
~~~~~~~

.. code-block:: xml
   :linenos:

   <oaire:citationEndPage>105</oaire:citationEndPage>
