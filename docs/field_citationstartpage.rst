27. Citation Start Page (R)
===========================

OAIRE Field
~~~~~~~~~~~
``oaire:citationStartPage``

Usage
~~~~~

The start page of the work published in a container (e.g. journal issue)

Example
~~~~~~~

.. code-block:: xml
   :linenos:

   <oaire:citationStartPage>100</oaire:citationStartPage>
