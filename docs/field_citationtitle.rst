24. Citation Title (R)
======================

OAIRE Field
~~~~~~~~~~~
``oaire:citationTitle``

Usage
~~~~~

The title name of the container (e.g. journal, book, conference) this work is published in.

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <oaire:citationTitle></oaire:citationTitle>
