24. Citation Title (R)
======================

``oaire:citationTitle``

Cardinality
~~~~~~~~~~~

*Recommended*

*Occurence: 0-1*

Definition and Usage Instruction
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The title name of the container (e.g. journal, book, conference) this work is published in.

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <oaire:citationTitle>some Journal Title</oaire:citationTitle>
