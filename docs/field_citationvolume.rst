25. Citation Volume (R)
=======================

``oaire:citationVolume``

Cardinality
~~~~~~~~~~~

*Recommended*

*Occurence: 0-1*

Definition and Usage Instruction
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The volume, typically a number, of the container (e.g. journal)

Example
~~~~~~~

.. code-block:: xml
   :linenos:

   <oaire:citationVolume>10</oaire:citationVolume>
