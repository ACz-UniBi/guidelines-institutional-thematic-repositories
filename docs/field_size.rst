20. Size (O)
^^^^^^^^^^^^

Unstructured size information about the resource (occurrences: 0-n).

**Allowed values, examples, other constraints**

Free text.

Examples: "15 pages", "6 MB"

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <datacite:sizes>
     <datacite:size>15 pages</datacite:size>
     <datacite:size>6 MB</datacite:size>
   </datacite:sizes>
